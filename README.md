# Linux-tools

This project includes all bash script to install tools on Linux system

Clone project

```
git clone 
```

## Docker Ecosystem

Install docker

```bash
docker/docker.sh
```

Install docker-compose

```bash
docker/docker-compose.sh
```

## CLI-Tools

Install aws-cli

```bash
cli-tools/awscli.sh
```

Install redis-cli

```bash
cli-tools/rediscli.sh
```
